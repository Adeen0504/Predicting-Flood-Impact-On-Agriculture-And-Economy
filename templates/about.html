Project Details

Objective

To predict flood impacts on agriculture, infrastructure, and economy in Telangana using Machine Learning models trained on historical data (2025-2026).

Dataset Overview

  • Time Period: 2025-2026
  • Total Records: {{ stats.records }} flood events
  • Districts Covered: All 33 districts
  • Crops Included: Rice, Cotton, Maize, Soybean, Vegetables, Chillies, Sugarcane
  • Features: 17 parameters including rainfall, soil, infrastructure, demographic data

Key Features

33 Districts 7 Crops 17 Features IMD Thresholds Soil Analysis Crop Suitability Economic Impact Humanitarian Market Analysis

7 Prediction Modules

Module 1: Flood Hazard IMD-based rainfall threshold analysis
Module 2: Crop Suitability Soil and climate compatibility
Module 3: Crop Damage ML prediction with 17 features
Module 4: Economic Loss Financial impact assessment
Module 5: Market Impact Price and supply forecasts
Module 6: Humanitarian Population and livestock affected
Module 7: Recommendations Actionable insights for farmers
